# WebGAL UI Theme Simulator

Standalone playground for drafting WebGAL UI themes without editing the engine or any job output.

Open `index.html` in a browser. Adjust parameters, then export:

- `theme.json`: stable parameter set for future theme tooling.
- `textbox.scss`: draft WebGAL template stylesheet output.

This simulator is intentionally separate from runtime code. After a theme is approved, copy the generated values into the project's theme library or a future theme generator.
